System Requirements for Automated Installations Using Media

To install by booting from AI media, the client systems must meet the following requirements:

Note - Any system that meets these requirements can be used, including laptops, desktops, virtual machines, and enterprise servers.

Table 4-1 Client Requirements

Requirement

Specifications

Memory

Minimum memory: 512 MB

Disk Space

Minimum disk space: 13 GB

Network Access

Client systems must be on a network, so that the system can access the following during the installation:

  • A DHCP server that provides network configuration information

  • An IPS repository that contains the packages to be installed on the client system

  • Optional: A custom manifest posted on an HTTP server
