Chapter 2

Introduction to the Automated Installer

If you want to install the OpenSolaris operating system (OS) on multiple client systems on a network, you can use the automated installer (AI) to accomplish that task. The automated installer performs essentially “hands-free” network installations of the OpenSolaris OS. You can install the OpenSolaris OS onto x86 systems or SPARC systems in a network.

Alternately, starting with the OpenSolaris 2010.03 release, you can simply download a SPARC or x86 AI image and burn it to media such as a CD, DVD, or a USB stick. Then, you can install the OpenSolaris OS directly on a client system by using this media.

  • If you want to install a few client systems directly from a CD or other media, see Chapter 4, Automated Installations Using Media.

    Note - If your SPARC client system does not have WANboot capability, use this method.

  • If you want to perform multiple network installations, review the following introduction to this tool.

  • If you have already set up your network to perform automated installations, you can go to the instructions for Chapter 6, How to Administer an Automated Install Server.
