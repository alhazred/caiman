Chapter 4

Automated Installations Using Media

Starting with the OpenSolaris 2010.03 release, you can do an automated installation of the OpenSolaris OS on SPARC or x86 client systems using media such as a CD, DVD, or a USB stick. You can install the client using either the default installation specifications provided in the media, or you can create and use a manifest with custom specifications.

Note - The client system must have network access. During the installation, the CD or other media will pull installation packages from an IPS repository on the network.
