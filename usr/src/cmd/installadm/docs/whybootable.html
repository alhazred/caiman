Why Install Using Media?

Automated installations using a CD or other media are particularly useful for the following reasons.

  • Automated installations using media save time. If you just want to install a few client systems, it may not be worth taking time to set up an install server and install service.

  • By installing using media, you can install to SPARC systems that do not have WAN Boot capability.

  • Automated installations using media is a fast, effective way to install clients when you don't need to customize the installation.

    Note - The installation specifications are provided in a manifest. You have the option of using the default manifest, or you could create a custom manifest.

  • The automated installation (AI) media that you create can be used as a rescue disk. Once you have created the CD, DVD, or USB stick, you can use it to perform simple “hands-off” automated installations to your x86 or SPARC client systems. The installations are non-interactive.
