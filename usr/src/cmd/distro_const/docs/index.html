Chapter 1

Introduction to the Distribution Constructor

The distribution constructor is a tool that application developers can use to build their own custom OpenSolarisTM image, which they can then distribute to their contacts and customers.

  • If you have not used the distribution constructor before, read the following introduction.

  • If you are ready to use the distribution constructor, go to Chapter 2, Design and Build OpenSolaris Images.

  • If you want to build a virtual machine, see the instructions at Chapter 3, x86: Design and Build a Virtual Machine.
