How to Review What You Have on Your Server

You can use the following installadm list command options to view information about your existing install services, associated install clients, and available manifests.

  • How to List Install Services on the Server

  • How to View Information for a Specified Install Service

  • How to List all Available Manifests

  • How to List Manifests Associated With an Install Service

  • How to List Clients Added to Your Install Services

  • How to List Clients Added to a Specific Install Service

How to List Install Services on the Server

The following command displays all of the enabled install services on your server.

# installadm list

In the following example, two install services, svcAI and svcAI-new, are found on the server.

# installadm list

Service Name Status       Arch  Port  Image Path
------------ ------       ----  ----  ----------
svcAI        on           x86   46501 /server/images/osol-1002-126-ai
svcAI-new    on           x86   46502 /server/images/osol-1003-111b2-ai

How to View Information for a Specified Install Service

The following command displays information about a specified install service, on your server, where svcname is the service name.

# installadm list -n svcname

In the following example, information about the specified install service, svcAI, is displayed.

# installadm list -n svcAI

Service Name Status       Arch  Port  Image Path
------------ ------       ----  ----  ----------
svcAI        on           x86   46501 /server/images/osol-1002-126-ai

How to List all Available Manifests

The following command lists all the manifests for all install services on your server.

# installadm list -m

In the following example, all the manifests are displayed for the 2 install services, svcAI and svcAI-new, found on the server.

# installadm list -m

Service Name Manifest
------------ --------
svcAI        devpublisher
             devpublisher3
             devpublisher4
             devpublisher5
             devpublisher6
svcAI-new    devpublisher7

How to List Manifests Associated With an Install Service

The following command displays information about all the manifests associated with a specified install service, svcname.

# installadm list -m -n svcname

In the following example, details are provided for all the manifests associated with the install service, svcAI-new.

# installadm list -m -n svcAI-new

Manifest      Criteria
--------      --------
devpublisher  arch = i86pc
              mac  = 01:C2:52:E6:4B:E0
              
devpublisher3 arch = i86pc
              mac  = 01:C2:52:E6:4B:E6 - 01:C2:52:E6:4B:E9

devpublisher4 arch = i86pc
              ipv4 = 192.168.168.151

devpublisher5 arch = i86pc
              mac  = 01:C4:52:E6:4B:E6 - 01:C4:52:E6:4B:E9
              ipv4 = 192.168.168.251

devpublisher6 arch = i86pc
              mac  = 01:C4:51:E6:4B:E6 - 01:C4:51:E6:4B:E9
              ipv4 = 192.168.168.012
              mem  = 2048 MB

How to List Clients Added to Your Install Services

The following command lists all the clients that have been added to the install services on a local server.

# installadm list -c

In the following example, all the clients are listed for the two install services, svcAI and svcAI-new, that are found on the server.

# installadm list -c

Service Name Client Address    Arch  Image Path
------------ --------------    ----  ----------
svcAI        01:C2:52:E6:4B:E1 x86   /server/images/osol-1002-126-ai
svcAI-new    08:00:27:8B:BD:71 x86   /server/images/osol-1003-111b2-ai
             01:C2:52:E6:4B:E0 x86   /server/images/osol-1002-126-ai

How to List Clients Added to a Specific Install Service

The following command lists all the clients that have been added to a specified install service, svcname.

# installadm list -c -n svcname

In the following example, one client has been added to the svcAI install service.

# installadm list -c -n svcAI

Service Name Client Address    Arch  Image Path
------------ --------------    ----  ----------
svcAI        01:C2:52:E6:4B:E1 x86   /server/images/osol-1002-126-ai
